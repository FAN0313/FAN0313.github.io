初中语文阅读题答题技巧

发表于2024-11-03|更新于2024-11-03
|字数总计:1.6k|阅读时长:4分钟|阅读量:

记叙文：

  • 记叙六要素

    时间，地点，人物，事情起因，经过，结果。

  • 记叙线索

    物线，事线，人线，地线，时线。

  • 记叙顺序

    顺序，倒序，插叙，补叙

    插叙作用：

    1. 结构上，使结构富于变化，避免平铺直叙。
    2. 内容上，交代了….（家境，缘由，内容等）。
    3. 写法上，为下文情节或刻画人物做铺垫，与…对比，突出…。
    4. 主题上，深化主题。
    5. 人物形象，刻画人物性格，丰富人物形象。
    6. 效果上，补充交代情节，使其富有波澜。

  • 记叙人称

    第一人称，第二人称，第三人称。

  • 写作手法

    • 修辞手法： 比喻，拟人，反复，夸张，排比，设问，反问，对偶。
    • 表现手法：铺垫，伏笔，前后照应，象征，衬托，对比，烘托，抑扬结合，托物言志，设置悬念，正侧面结合，以小见大，虚实结合，联想想象，借景抒情。
    • 表达方式：记叙，描写，说明，议论，抒情
    • 描写手法：
      • 按照描写对象分
        1. 人物描写：外貌，语言，动作，神态，心理
        2. 环境描写：自然环境描写，社会环境描写
      • 按照描写角度分
        1. 正面描写（直接描写）
        2. 侧面描写
        3. 细节描写

  • 记叙语言

    • 锤炼词语
    • 修辞手法
    • 句式选择
    • 整体语言风格

赏析句子

  • 1.品味字词

​ 【答题格式】…词是…意思，在文中的意思是…，生动形象地写出了…（事物|人物|景物）的…特点或内容，表达了作者…的情感。 (如果换成…词，就没有这种效果了，所以不能换)

  • 2.赏析句子

    • 修辞角度：（修辞手法+描写内容+情感）

    • 词语运用：（动词、叠词、拟声词……)

    • 句式：疑问句（吸引读者，制造悬念）；设问句（提醒注意，引发思考）；反问句（加强语气）；感叹句（强烈抒情）；长短句（节奏变化，富有音韵美）

    • 描写角度：

      • 从人物描写角度赏析

      • 环境描写角度赏析环境描写的作用：

        1. 交代故事发生的环境、时间，渲染……气氛

        2. 烘托人物……心情/情感/形象，暗示人物的前途命运

        3. 推动故事情节发展,为下文情节做铺垫

        4. 暗示(深化)主题

        5. 借景抒情,情景交融

      • 从作者感官角度（视觉、听觉、嗅觉等）

      • 表达方式：

          1. **议论：**点明事件意义和文章主旨，起统领全篇、画龙点睛的作用。
  2. **抒情：**使文章具有感染力
  3. **描写：**使人、景、物再现得栩栩如生，使文章更加生动形象
  4. **记叙：**使读者对人物活动和事件发展、全貌有清晰的了解。
  5. **说明**

标题分析

1.标题含义

  • A 表层含义：直接理解标题的字面意思。
  • B 深层含义：
    • 背景：标题反映的时代、环境等背景信息。
    • 线索：贯穿全文的核心元素或事件。
    • 修辞：如比喻、象征等手法赋予的深层意义。
    • 内容：标题所涵盖的主要故事或主题。
    • 主旨：文章要表达的核心思想或情感。
    • 人物：标题与主要人物的关系或象征。

2. 标题作用

  • A 内容：概括文章主要内容。
  • B 主旨：揭示或暗示文章的中心思想。
  • C 线索：作为全文的线索，串联故事情节。
  • D 修辞：运用修辞增强标题的吸引力。
  • E 人物：与主要人物的形象或命运相关。
  • F 表达效果：如引发读者兴趣、设置悬念等。
  • G 情感：传达作者的情感倾向。

段落作用

开头段作用

  • 内容上：开篇点题，引出叙述内容。
  • 结构上：
    • 总领全文，引出下文。
    • 照应标题。
    • 吸引读者阅读兴趣。

中间段作用

  • 内容上：
    • 概括具体内容。
    • 点明中心，表达情感。
  • 结构上：
    • 承上启下。
    • 照应标题、前文。

结尾段作用

  • 内容上：
    • 概括具体内容。
    • 点明中心，升华主题。
  • 结构上：
    • 总结全文。
    • 照应标题、前文。
    • 设置悬念，引发思考。

说明文部分

说明文分类与顺序

  • 分类：
    • 事物说明文：介绍事物形体特征。
    • 事理说明文：介绍事物本身的道理和内在规律。
  • 顺序：
    • 时间顺序。
    • 空间顺序。
    • 逻辑顺序。

常考题型及解答

题型1：段落顺序能否调换？

  1. 表态：不能调换。
  2. 概括内容：分别概括两个段落的内容。
  3. 说明顺序：指出全文或段落间的说明顺序或前后照应关系。
  4. 总结：如果调换，前后不对应，不符合人们认识事物的规律。

说明方法：

  • 举例子：举…的例子，具体说明……的……特点，从而使说明更具体，有说服力。
  • 列数字：用具体的数据说明了……，使说明更准确、更有说服力。
  • 作比较：把……和……作比较，突出强调说明了……事物的…特征/事理。
  • 打比方：把……比作……。生动形象地说明亦物的……特征/事理，增强了文章的趣味使要说明对象形象可感。
  • 分类别：对……事物的……特征/事理分门别类加以说明，显得条理清晰，一目了然。
  • 摹状貌：生动形象地说明了……。使说明对象具体可感。
  • 作引用：引用……突出了……。增强了文章的说服力和可信度。
  • 下定义：更科学、准确、简明地揭示了……的本质特点，便于读者对说明对象获得清晰的概念。

说明文结构：

​ 总分式，并列式，递进式

说明语言：

​ 准确严谨，生动平实

题型2：加点词能否删掉/替换？

  • 不能删/换。
  • 解词：解释词语原意。
  • 带词解句：说明该词在句子中的具体作用。
  • 去词/换词解句：去掉或替换后句子意思的变化。
  • 总结：该词体现了说明文语言的准确性（或生动性），所以不能删/换。

题型3：说明文第一段的作用？

  • 引出说明对象/内容。
  • 吸引读者兴趣。
  • 照应题目。
  • 引出下文（作为某种说明方法）。
